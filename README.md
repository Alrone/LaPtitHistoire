La Ptit Histoire
=============================

Mini-jeu écrit en Symfony2

-----------------------------

Comment ca marche :

IL y a 6 éléments (eau, soleil, graines, pipeau, temps, N).

Pour chaque scène :

* Un élément permet de passer à la scène suivante.
* Un élément fait reculer à la scène précédente
* Les autres éléments ne font rien

Quand le tableau est finit, c'est la "fin".

-----------------------------

- Comment couper cette satanée musique :

Cliquer sur la note de musique à droite
